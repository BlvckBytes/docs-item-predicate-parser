# Has-Name Predicate

Matches against items which have a special name set, be it a custom-name or a display-name.

| Language | Identifier |
|:--------:|:----------:|
| en_us | Has-Name |
| en_gb | Has-Name |
| de_de | Hat-Name |
| de_at | Hat-Name |
| de_ch | Hat-Name |
| zh_cn | Has-Name |
| tr_tr | Has-Name |
| fr_fr | Has-Name |


## Examples

Input: Has-Name\
Result: All items which have a special name set
